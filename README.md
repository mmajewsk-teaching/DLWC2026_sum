### Useful commands

#### If you installed this repo's pixi env, you can source env like this:

```
source /path_to_your_local_repo/DLWC2026_sum/.pixi/envs/default
```

#### Alternatively inside the repo:

```
pixi shell
```

```
source /home/dokt/majewski/Public/mlenv204pixi/activate.sh
```
